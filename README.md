# 🚀 Apprentissage React & TypeScript - Exercices Pratiques

Ce dépôt rassemble une série d'exercices progressifs conçus pour maîtriser les concepts fondamentaux et intermédiaires de **React** avec **TypeScript**. L'objectif est de passer d'une compréhension théorique à une posture de développeur autonome et professionnel.

## 🛠️ Architecture du Dépôt

Pour maximiser l'apprentissage, ce dépôt suit une structure stricte :
* **Branche `main` :** Contient uniquement les énoncés des exercices dans ce README et la configuration de base du projet (vide).
* **Branches dédiées (`exercice-1`, `exercice-2`, etc.) :** Contiennent le code de la solution validé et optimisé (Clean Code, typage TypeScript rigoureux, accessibilité).

## 🏃‍♂️ Liste des Exercices

### 📦 Module 1 : Les Props (Communication entre composants)

#### Exercice 1 : Le Composant `Button` Typé (Débutant)
* **Énoncé :** Créer un composant `Button` réutilisable qui accepte une prop `label` (string) obligatoire. Le composant doit être correctement typé avec une interface TypeScript et appelé plusieurs fois dans `App.tsx` avec des textes différents.
* **Branche solution :** `exercice-1`

#### Exercice 2 : Props optionnelles et Valeurs par défaut (Débutant +)
* **Énoncé :** Ajouter une prop optionnelle `variant` au composant `Button` pouvant accepter uniquement les valeurs `"primary"` ou `"secondary"`. Si elle n'est pas fournie, elle doit valoir `"primary"` par défaut. Afficher trois boutons dans `App.tsx` pour tester tous les cas.
* **Branche solution :** `exercice-2`
